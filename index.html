An awesome arcade shoot'em'up. Shoot your way through a bunch of action packed missions. Free prisoners of wars from endless torture. Fly attack helicopters, behemoth gunships and armed transports. Over time we'll add more missions, more helicopters, more guns, more action and more everything.

Backer interest will decide the exact amount of features upon release. If interest is high enough, we'll even set up pre-orders for the OUYA release which will include all future content. Planned features:

  • Multiple helicopter types:
    • Light attack helicopter
    • Heavy attack helicopter
    • Armed transport helicopter
  • Varied missions
    • Search and destroy
    • Convoy escorts
    • Rescue missions
  • Big guns
  • Awesome 3D graphics
  • Head-to-head multiplayer

FAQ

What platforms will the game be released on?
First it will be released on the OUYA, but we are developing it in a cross-platform manner. If the game generates enough interest, we will look into releasing it on Steam and possibly other consoles.
